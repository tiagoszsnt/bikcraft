# 🚲 Bikcraft — Projeto Origamid

Projeto desenvolvido durante o curso da Origamid com foco em criação de um site institucional moderno, aplicando conceitos de HTML, CSS e JavaScript.

🔗 **Acesse o projeto online:**  
👉 [tiagoszsnt.github.io/bikcraft](https://tiagoszsnt.github.io/bikcraft/)

---

## 📋 Sobre o Projeto

O **Bikcraft** é um site institucional de uma empresa fictícia de bicicletas elétricas, com foco em apresentar produtos, planos e informações de forma clara e atrativa.

O projeto foi desenvolvido com atenção à **responsividade**, **organização de código** e **boas práticas de front-end**.

---

## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## 🎯 Funcionalidades

- Layout totalmente responsivo  
- Navegação entre seções  
- Interações com JavaScript (em desenvolvimento)  
- Estrutura organizada e semântica  

---

## 📚 Aprendizados

- Estruturação de páginas com HTML semântico  
- Estilização com CSS moderno (Flexbox e Grid)
